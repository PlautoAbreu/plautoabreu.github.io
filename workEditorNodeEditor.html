AI NODE EDITOR

The AI Node Editor is a visual editor to facilitate the construction of Behaviour Trees using my BT AI Library.

The editor/API currently support Selector/Sequence/Inverter/Behaviour/Condition Nodes.

The estimated launch date on the Asset Store is first quarter of 2017.

Date: February 2017 - Engine: Unity

ABOUT

❗ Custom Editor Window for Unity
❗ Persistant storage using ScriptableObjects.
❗ Easy to connect, disconnect, delete, replace.
❗ Behaviours and Conditionals nodes can run custom functions.


Both the editor and the API still under development. But a beta version is avaliable under request.
