CARD GAME VR (FrameWork)

This Prototype Ideia was inspired by the VR Trend, this is more of a Framework for any type of card game you want to implement in VR.

Here you look trough the eyes of the player, you can control the camera simulating head movement and can interact with the cards using your eye focus and both hands, one hand to hold the cards, the other to play them.

Date: June 2016 - Platform: PC

ABOUT

❗ VR Support
❗ Support for both Hand Controllers.
❗ Multiplayer.
❗ Custom Matchmaking system
❗ Custom C# Server


I decided to implement a plain C# server side to handle the matchs and the machmaking, letting the client side independent from any game logic. The integration with the Vive controller is holding the implementation since I don't have acess to them yet.
