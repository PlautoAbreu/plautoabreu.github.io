ETERNAL GRAVITY

Eternal Gravity was one one my first games, it was done for iOS using Xcode and UIKIT since there was no GameKit yet.

The game ideia was to be a multiplayer clicking game, where you would click the screen to aquire mass and upgrade your "star". You would then use the mass aquired to purchase upgrades that would allow you to gain mass faster and to contribute to your friends star mass as well.

ps. The lines are created with the touches.

Date: March 2013 - Platform: iOS

ABOUT

❗ Clicking game !
❗ Scrollable Menus.
❗ Leaderboards.

Although the game development was in a considerably advanced stage, I was too inexperienced to implement the multiplayer part. Nowadays the game is too outdated, but I still like the ideia of a multiplayer clicker game, ho knows, someday!
